## Who This Guide Is For

You want to build a website but you don't know how to code. That's fine. AI tools can build the entire thing for you — you just need to know **what to tell them** and **how to ship it**.

By the end of this guide, you'll have a live website on the internet with your own domain name. No coding required.

## Step 1: Decide What to Build (5 Minutes)

Before opening any AI tool, answer three questions:

**1. Who is this for?**
Examples: "People looking to hire me as a designer" or "Customers who want to buy my course"

**2. What's the one thing they should do?**
Examples: "See my work and contact me" or "Buy my course and get access"

**3. What pages do you need?**
Most websites only need 3-5 pages. Start simple.

### Example Answers

| Question | Portfolio Site | Course Site | Restaurant |
|----------|---------------|-------------|------------|
| Who? | Employers | Students | Hungry people |
| What? | See my work, hire me | Buy my course | See menu, order |
| Pages | Home, Projects, Contact | Home, Courses, Checkout | Home, Menu, Order |

## Step 2: Build with AI (30-60 Minutes)

You have two free options. Both build real, working websites.

### Option A: Lovable (Recommended)

Lovable builds full websites with a chat interface. You describe what you want, it builds it.

**How to use Lovable:**

1. Go to **lovable.dev**
2. Click "Sign up" (free with Google or GitHub)
3. Click "New Project"
4. Type your first prompt — this is the most important step

**Your first prompt should be detailed:**

```
Build me a portfolio website for a freelance graphic designer.

Pages needed:
1. Home page with hero section (name: "Sarah Chen", tagline: "Visual Designer & Brand Strategist")
2. Projects page showing 6 projects in a grid layout
3. About page with bio and skills
4. Contact page with a form (name, email, message)

Design style:
- Dark theme with white text
- Modern, clean, minimalist
- Use the color #3ad29f as the accent color
- Lots of whitespace between sections
- Font: Inter for body, bold for headings

Make it fully responsive (works on phone, tablet, and desktop).
```

5. Lovable generates the entire website in 2-5 minutes
6. Preview it in the browser — click through every page
7. Fix things by typing: "The contact form needs validation" or "Make the hero section bigger"

**When to stop prompting Lovable:**
Stop when you have:
- All pages working
- Navigation between pages works
- It looks good on mobile
- The content is correct

**Do NOT try to build complex features in Lovable.** That's what Step 3 is for.

### Option B: Bolt.new

Bolt runs a code editor directly in your browser. Good for quick prototypes.

1. Go to **bolt.new**
2. Describe your project
3. Edit the code directly in the browser
4. When satisfied, connect to GitHub and push

## Step 3: Push to GitHub (10 Minutes)

GitHub is where your code lives safely. Think of it as Google Drive for code.

**If you used Lovable:**
1. In Lovable, click the GitHub icon (top right)
2. Click "Push to GitHub"
3. It creates the repository automatically
4. Done — your code is saved

**If you used Bolt.new:**
1. Click the GitHub icon in Bolt
2. Authorize the connection
3. Bolt creates the repo and pushes the code

**Why GitHub matters:**
- Your code is backed up forever
- You can deploy from GitHub to Vercel
- You can collaborate with others
- You can track changes

## Step 4: Deploy to Vercel (5 Minutes)

Vercel is a hosting platform that makes your website live on the internet. It's free for personal projects.

### First Time Setup

1. Go to **vercel.com**
2. Click "Sign up" (use the same GitHub account from Step 3)
3. Click "Add New Project"
4. Find your repository (the one Lovable or Bolt created)
5. Click "Import"
6. Click "Deploy"
7. Wait 30 seconds
8. Your website is live at `https://your-project.vercel.app`

### Add Your Custom Domain

Instead of `your-project.vercel.app`, you want `yourname.com`.

1. Buy a domain from Namecheap, Google Domains, or Cloudflare (costs ~$10/year)
2. In Vercel → Settings → Domains
3. Type your domain name (e.g., `sarahchen.design`)
4. Vercel shows you DNS records to add
5. Go to your domain registrar (where you bought the domain)
6. Add the DNS records Vercel showed you
7. Wait 5-30 minutes
8. Your site is live at `https://sarahchen.design`

---

## Step 5: Set Up Google Search Console (10 Minutes)

This tells Google your website exists so people can find it.

1. Go to **search.google.com/search-console**
2. Click "Add Property"
3. Enter your full URL (e.g., `https://sarahchen.design`)
4. Verify ownership (Vercel can help with this)
5. Submit your sitemap (Vercel generates this automatically)
6. Done — Google will index your site within 24-48 hours

## Common Questions

**"I don't know what React or Next.js are — does that matter?"**
No. React and Next.js are programming frameworks that Lovable and Bolt use behind the scenes. You don't need to learn them. You just describe what you want, and the AI builds it using these tools.

**"What if I want to change something later?"**
Go back to Lovable and tell it what to change. Or hire a developer on Fiverr to make specific edits.

**"How much does this cost?"**
- Lovable: Free tier available
- Bolt.new: Free tier available
- GitHub: Free for public repos
- Vercel: Free for personal projects
- Domain: ~$10/year
- **Total: ~$10/year for a live, professional website**

**"Can I add a blog or e-commerce?"**
Yes. Lovable supports both. For a blog, tell Lovable "Add a blog page where I can write posts." For e-commerce, tell it "Add a shop page with Stripe checkout."

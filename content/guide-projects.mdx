## Build Real Things

The best way to learn is to build. Here are 10 projects you can build using AI tools — no coding required. Each project teaches you something new.

---

## Beginner Projects (1-2 Days Each)

### Project 1: Personal Portfolio

**What you build:** A one-page website showing your work and how to contact you.

**What to tell Lovable:**
```
Build me a one-page portfolio website.

Sections:
1. Hero with my name "Your Name" and tagline "Web Designer & Developer"
2. Projects section showing 4 projects with images and descriptions
3. About section with a photo and short bio
4. Contact form (name, email, message)

Design: Dark theme, clean, modern, lots of whitespace.
Color: #3ad29f as accent.
Mobile responsive.
```

**Skills you learn:** Describing layouts, project structure, responsive design.

---

### Project 2: Blog Site

**What you build:** A website where you can write and publish articles.

**What to tell Lovable:**
```
Build me a blog website.

Features:
1. Home page showing 6 blog posts in a grid
2. Individual blog post page with title, date, and content
3. About page
4. Dark theme, clean typography

I want to write posts in markdown files.
```

**Skills you learn:** Content management, dynamic pages, markdown.

---

### Project 3: Landing Page

**What you build:** A single page that sells a product or service.

**What to tell Lovable:**
```
Build me a landing page for a fictional course called "AI Mastery".

Sections:
1. Hero: "Master AI in 30 Days" with a "Buy Now" button
2. Features: 3 things you'll learn
3. Testimonials: 3 fake reviews
4. Pricing: One plan at $99
5. FAQ: 4 common questions

Design: Bold, modern, dark background, green accent.
```

**Skills you learn:** Sales pages, conversion design, call-to-action buttons.

## Intermediate Projects (3-5 Days Each)

### Project 4: AI Chatbot

**What you build:** A chat interface that talks to an AI.

**What to tell Lovable:**
```
Build me an AI chatbot.

Features:
1. Chat interface with message bubbles
2. User messages on right, AI messages on left
3. Loading spinner while AI thinks
4. Connects to Groq API

Technical: Use environment variable for GROQ_API_KEY.
Never hardcode the API key.
```

**What you need:** A free Groq API key (see the Build AI guide).

**Skills you learn:** API integration, environment variables, loading states.

---

### Project 5: Task Manager

**What you build:** A to-do list app with user accounts.

**What to tell Lovable:**
```
Build me a task manager app.

Features:
1. User signup and login
2. Create tasks with title, description, and due date
3. Mark tasks as complete
4. Delete tasks
5. Dashboard showing all tasks

Use Supabase for the database and authentication.
```

**What you need:** A free Supabase account.

**Skills you learn:** User authentication, databases, CRUD operations.

---

### Project 6: E-commerce Store

**What you build:** A shop where people can buy products.

**What to tell Lovable:**
```
Build me a simple e-commerce store.

Features:
1. Product listing page with images and prices
2. Individual product pages
3. Shopping cart
4. Checkout with Stripe

Use Supabase for products and Stripe for payments.
```

**What you need:** Stripe account (free to start).

**Skills you learn:** Payment processing, product management, cart functionality.

---

## Advanced Projects (1-2 Weeks Each)

### Project 7: AI Content Generator

**What you build:** A tool that generates blog posts, emails, or social media content using AI.

**What to tell Lovable:**
```
Build me an AI content generator.

Features:
1. Choose content type (blog post, email, tweet)
2. Enter a topic
3. AI generates the content
4. Copy to clipboard button
5. History of generated content

Use Groq API with a custom system prompt for each content type.
```

**Skills you learn:** System prompts, content types, API integration.

---

### Project 8: Dashboard

**What you build:** An analytics dashboard with charts and graphs.

**What to tell Lovable:**
```
Build me an analytics dashboard.

Features:
1. Sidebar navigation
2. Main area with charts showing:
   - Revenue over time (line chart)
   - Users by country (bar chart)
   - Sales by category (pie chart)
3. Date range picker
4. Dark theme, professional look

Use fake data for now.
```

**Skills you learn:** Data visualization, charts, dashboard layouts.

---

### Project 9: AI Writing Assistant

**What you build:** A tool that helps you write better content.

**What to tell Lovable:**
```
Build me an AI writing assistant.

Features:
1. Text editor area on the left
2. AI suggestions panel on the right
3. Buttons: "Improve", "Fix Grammar", "Make Shorter", "Make Longer"
4. Click a button → AI rewrites the text

Use Groq API. Each button sends a different system prompt.
```

**Skills you learn:** Text manipulation, multiple system prompts, UI design.

---

### Project 10: Full SaaS App

**What you build:** A complete software-as-a-service application with user accounts, billing, and multiple features.

**What to tell Lovable:**
```
Build me a SaaS app for managing client projects.

Features:
1. User authentication (signup, login, logout)
2. Dashboard showing project overview
3. Create, edit, delete projects
4. Each project has tasks with status (todo, in progress, done)
5. Team invites (invite others to collaborate)
6. Settings page
7. Stripe subscription billing ($19/month)

Use Supabase for database and auth, Stripe for payments.
```

**Skills you learn:** Full-stack architecture, multi-tenancy, billing.

## How to Use These Projects

1. **Start with Project 1** — even if you're experienced
2. **Build each project fully** — don't skip steps
3. **Deploy every project** — a project on Vercel is worth 10x more than one on your laptop
4. **Add to your portfolio** — each project becomes a showcase piece
5. **Write about it** — blog posts about what you built = learning reinforcement

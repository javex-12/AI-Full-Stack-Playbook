## The Architecture: Why Pagefind

Traditional search requires a running server — Elasticsearch, Algolia, Meilisearch — each with hosting costs, API key management, and latency from network round trips. For a static site, this is overkill.

**Pagefind is different.** It runs a compiled Rust binary at build time, parsing your HTML and chunking keywords into tiny cached fragments. When a user searches, the browser downloads *only* the specific fragments needed — usually under 30KB total. Zero backend. Zero infrastructure. Zero latency.

**Traditional Search (Algolia/Elasticsearch):**
- User types → Network request → Server queries index → Response → Render
- Latency: ~50-200ms round trip

**Pagefind (Static):**
- User types → Local WASM index → Instant results → Render
- Latency: ~1-5ms local execution

## Step 2: Content Targeting

By default, Pagefind indexes your entire page — including navbars, footers, and sidebars. This creates noisy results. You need to tell Pagefind exactly what content to index.

### Mark Indexable Content

Add `data-pagefind-body` to your main content wrapper. Only content inside this attribute gets indexed.

**Tell your AI tool:**
"Add data-pagefind-body attribute to the main content element in the layout. The sidebar and footer should NOT have this attribute — they won't be searchable. Only the article content inside main should be indexed."

### Fine-Grained Control

| Attribute | Purpose | Example |
|-----------|---------|---------|
| `data-pagefind-body` | Mark indexable content area | `<main data-pagefind-body>` |
| `data-pagefind-weight="2"` | Boost importance (default: 1) | `<h1 data-pagefind-weight="2">Title</h1>` |
| `data-pagefind-weight="0.5"` | Reduce importance | `<aside data-pagefind-weight="0.5">Sidebar</aside>` |
| `data-pagefind-exclude` | Exclude element and children | `<div data-pagefind-exclude>Hidden content</div>` |
| `data-pagefind-meta="title"` | Extract metadata for results | `<h1 data-pagefind-meta="title">Page Title</h1>` |
| `data-pagefind-filter="type:guide"` | Add filterable facets | `<article data-pagefind-filter="type:guide">` |

### Excluding Navigation from Results

**Tell your AI tool:**
"Add data-pagefind-exclude to navigation elements so they don't appear in search results. The nav should be visible but not searchable."

<script>
  // =============================================
  // Pagefind Search Integration
  // =============================================

  let pagefind: any = null;
  let activeIndex = -1;
  let currentResults: HTMLElement[] = [];

  const overlay = document.getElementById('searchOverlay')!;
  const input = document.getElementById('searchInput') as HTMLInputElement;
  const resultsContainer = document.getElementById('searchResults')!;
  const status = document.getElementById('searchStatus')!;

  // -------------------------------------------
  // Open / Close
  // -------------------------------------------
  function openSearch() {
    overlay.classList.add('open');
    input.focus();
    input.value = '';
    resultsContainer.innerHTML = '';
    status.classList.remove('visible');
    activeIndex = -1;
    document.body.style.overflow = 'hidden';
  }

  function closeSearch() {
    overlay.classList.remove('open');
    document.body.style.overflow = '';
    input.blur();
  }

  // -------------------------------------------
  // Cmd+K / Ctrl+K to open
  // -------------------------------------------
  document.addEventListener('keydown', (e) => {
    if ((e.metaKey || e.ctrlKey) && e.key === 'k') {
      e.preventDefault();
      if (overlay.classList.contains('open')) {
        closeSearch();
      } else {
        openSearch();
      }
    }

    if (e.key === 'Escape' && overlay.classList.contains('open')) {
      closeSearch();
    }
  });

  // Click overlay background to close
  overlay.addEventListener('click', (e) => {
    if (e.target === overlay) closeSearch();
  });

  // -------------------------------------------
  // Load Pagefind on first focus
  // -------------------------------------------
  let pagefindLoaded = false;

  async function loadPagefind() {
    if (pagefindLoaded) return;
    try {
      pagefind = await import('/pagefind/pagefind.js');
      await pagefind.init();
      pagefindLoaded = true;
    } catch (err) {
      console.error('Pagefind not available. Run build first.', err);
    }
  }

  // -------------------------------------------
  // Debounce utility
  // -------------------------------------------
  function debounce(fn: Function, ms: number) {
    let timer: ReturnType<typeof setTimeout>;
    return (...args: any[]) => {
      clearTimeout(timer);
      timer = setTimeout(() => fn(...args), ms);
    };
  }

  // -------------------------------------------
  // Search execution
  // -------------------------------------------
  const executeSearch = debounce(async (query: string) => {
    if (!query || !pagefind) {
      resultsContainer.innerHTML = '';
      status.classList.remove('visible');
      currentResults = [];
      activeIndex = -1;
      return;
    }

    status.textContent = 'Searching...';
    status.classList.add('visible');

    const searchInstance = await pagefind.search(query);
    const count = searchInstance.results.length;

    status.textContent = count > 0
      ? `${count} result${count !== 1 ? 's' : ''}`
      : 'No results found';

    resultsContainer.innerHTML = '';
    currentResults = [];
    activeIndex = -1;

    if (count === 0) {
      resultsContainer.innerHTML = `
        <div class="search-no-results">
          No results for "<strong>${escapeHtml(query)}</strong>"
        </div>
      `;
      return;
    }

    // Load top 8 results
    const topResults = searchInstance.results.slice(0, 8);

    for (const item of topResults) {
      const data = await item.data();
      const el = document.createElement('a');
      el.className = 'search-result';
      el.href = data.url;
      el.setAttribute('role', 'option');

      // Extract section from meta if available
      const section = data.meta?.section || '';

      el.innerHTML = `
        ${section ? `<div class="search-result-section">${escapeHtml(section)}</div>` : ''}
        <div class="search-result-title">${escapeHtml(data.meta?.title || 'Untitled')}</div>
        <div class="search-result-excerpt">${data.excerpt}</div>
      `;

      resultsContainer.appendChild(el);
      currentResults.push(el);
    }
  }, 200);

  // -------------------------------------------
  // Input handler
  // -------------------------------------------
  input.addEventListener('input', () => {
    executeSearch(input.value.trim());
  });

  // -------------------------------------------
  // Keyboard navigation
  // -------------------------------------------
  input.addEventListener('keydown', (e) => {
    if (!currentResults.length) return;

    if (e.key === 'ArrowDown') {
      e.preventDefault();
      activeIndex = Math.min(activeIndex + 1, currentResults.length - 1);
      updateActiveResult();
    }

    if (e.key === 'ArrowUp') {
      e.preventDefault();
      activeIndex = Math.max(activeIndex - 1, -1);
      updateActiveResult();
    }

    if (e.key === 'Enter' && activeIndex >= 0) {
      e.preventDefault();
      const link = currentResults[activeIndex].getAttribute('href');
      if (link) window.location.href = link;
    }
  });

  function updateActiveResult() {
    currentResults.forEach((el, i) => {
      el.classList.toggle('active', i === activeIndex);
    });

    if (activeIndex >= 0 && currentResults[activeIndex]) {
      currentResults[activeIndex].scrollIntoView({ block: 'nearest' });
      input.setAttribute('aria-activedescendant', `result-${activeIndex}`);
    } else {
      input.removeAttribute('aria-activedescendant');
    }
  }

  // -------------------------------------------
  // Utility
  // -------------------------------------------
  function escapeHtml(str: string): string {
    const div = document.createElement('div');
    div.textContent = str;
    return div.innerHTML;
  }

  // -------------------------------------------
  // Init: load Pagefind when search opens
  // -------------------------------------------
  input.addEventListener('focus', loadPagefind, { once: true });
</script>
```

### Common Issues

| Symptom | Cause | Fix |
|---------|-------|-----|
| Search returns nothing | Pagefind not run after build | Add `&& pagefind --site dist` to build script |
| Navigation items in results | Missing `data-pagefind-exclude` | Add exclusion attribute to nav/footer |
| Results show raw HTML tags | Content not properly rendered | Ensure `data-pagefind-body` is on rendered content |
| Search is slow on first query | WASM not cached | Normal for first load; subsequent queries are instant |
| Index is huge (>500KB) | Too much content indexed | Use `data-pagefind-exclude` to trim non-essential pages |

## Key Takeaways

| Concept | Mental Model | Senior Insight |
|---------|-------------|----------------|
| Static Search | WASM binary runs locally in browser | Zero server cost, zero latency, zero API keys |
| Content Targeting | `data-pagefind-body` marks indexable regions | Exclude nav/footer to avoid noisy results |
| Weight & Metadata | Boost importance, extract titles | Use `data-pagefind-weight` to rank critical content higher |
| Command Palette | Cmd+K opens search overlay | Keyboard navigation (↑↓ Enter ESC) is expected UX |
| Debouncing | Wait 200ms after last keystroke | Prevents searching on every single character |
| Lazy Loading | Import Pagefind WASM on first focus | Zero JS cost until the user actually wants to search |
